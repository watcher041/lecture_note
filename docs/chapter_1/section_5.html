回転運動について

　ここからは、どのように回転運動を扱うかを述べていくことにする。そもそも回転というのは、ある一点を軸にその周囲を運動することを言う。その様子を記述するとき、ちょうど二体問題で扱った極座標 (r,θ) が便利であるため、試しに運動方程式

mdt2d2r​=F

を変形してどのように記述されるかを見てみよう。ここで、r は質点の位置、F は質点に働く力、 m は質点の質量としている 。まず、直交座標系 (x,y) を用いて

mdt2d2x​=Fx​、mdt2d2y​=Fy​

となるが、ここで直交座標系と極座標との間では以下の関係式が成り立つ。

x=rcosθ、y=rsinθ

この関係式というのは、座標を回転させたときに得られる変換式の一つの利用例となっている。実際、座標系 (x,y) を原点Oを軸に角度 θ だけ回転させて座標系 (X,Y) に変換したものとすると、

X=xcosθ+ysinθ、Y=−xsinθ+ycosθ

となるが、この逆変換を求めると

x=Xcosθ−Ysinθ、y=Xsinθ+Ycosθ

であり、X=r,Y=rθ​=0 とすると、極座標の形になること分かる。同様に、力や速度、加速度も同じ変換が適用されるため、

Fx​=Fr​cosθ−Frθ​​sinθ、Fy​=Fr​sinθ+Frθ​​cosθ vx​=vr​cosθ−vrθ​​sinθ、vy​=vr​sinθ+vrθ​​cosθ ax​=ar​cosθ−arθ​​sinθ、ay​=ar​sinθ+arθ​​cosθ

も成り立つ。このことを踏まえたうえで、まず各方向の速度を極座標の形であらわすと

vx​=dtdx​=dtdr​cosθ−rdtdθ​sinθ vy​=dtdy​=dtdr​sinθ+rdtdθ​cosθ

となるが、これを先ほどの速度の極座標への変換式に代入して vr​,vrθ​​ を求めると以下の式が得られる。

vr​=dtdr​、vrθ​​=rdtdθ​

この形については、以下のように各方向の微小変化を見ることで正しいものと確認できる。

次に、加速度についても同様に

ax​=dt2d2x​=dt2d2r​cosθ−2dtdr​dtdθ​sinθ−rdt2d2θ​sinθ−r(dtdθ​)2cosθ ay​=dt2d2y​=dt2d2r​sinθ+2dtdr​dtdθ​cosθ+rdt2d2θ​cosθ−r(dtdθ​)2sinθ

となるため、こちらも先ほどの変換式に代入して ar​,arθ​​ を求めると

ar​=dt2d2r​−r(dtdθ​)2、arθ​​=2dtdr​dtdθ​+rdt2d2θ​

となるが、力の変換式も踏まえて極座標での運動方程式は以下の通りになっていることが分かる。

m[dt2d2r​−r(dtdθ​)2]=Fr​、m(2dtdr​dtdθ​+rdt2d2θ​)=Frθ​​

これらの式を詳しく見ていくと、動径方向については

mdt2d2r​=Fr​+fr,θ​、fr,θ​=mr(dtdθ​)2

というように、通常の運動方程式にあたかも力 fr,θ​ （遠心力）が遠ざかる方向に加わった形になっている。次に、角度方向については二体問題と同じように r をかけて整理すると

dtdL​=rFrθ​​、L=mr2dtdθ​

というように L を用いて書ける。ここで回転する条件を考えたとき、角度方向に動いていれば回転することから角度の時間微分（角速度）が常に0でないものと思われる。そこで、角速度を ω として先ほどの式を表記し直すと

fr,θ​=mrω2、L=mr2ω、ω=dtdθ​

となるが、このうち L は従来の運動量が質量（慣性質量）と速度の掛け算であったことを踏まえ

L=Iω、I=mr2

というように I（慣性モーメント）を用いると、L は角度方向の運動量（角運動量）として扱うことができる。すると、rFrθ​​ は回転運動を変化させる量として N（モーメント）としてみなすことができ、

dtdL​=N、L=Iω、N=rFrθ​​

というように回転の運動方程式として記述される。ここで、N について詳しく見ていくために直交座標系で書くと

N=rFrθ​​=rcosθFy​−rsinθFx​=xFy​−yFx​

となるが、これは外積と呼ばれる関係になっていることが予想される。例えば、ベクトル a,b が以下の図のようになっており a=(ax​,ay​,0)、b=(bx​,by​,0)

とすると、外積 c は以下の形で表される。

c=a×b=(0,0,ax​by​−ay​bx​)

試しに a=r,b=F として外積をとると、以下の通り z 成分が先ほどの N と一致していることが分かる。

r×F=(0,0,xFy​−yFx​)=(0,0,N)

そのため、回転方向については z 成分に関する方程式になっていると考えられる。そこで、運動方程式

mdtdv​=F

に外積をとることで一般的に回転の運動方程式がどのようになるかを見てみることにする。まず、両辺に r と外積をとると

m(r×dtdv​)=N、N=r×F

というように右辺がモーメント N となるので、左辺を変形してみると

(r×dtdv​)=dtd​(r×v)−(v×v)=dtd​(r×v)

となるので以下の通りになる。

mdtd​(r×v)=N

そして、左辺を極座標で整理することで

m(r×v)=(0,0,mxdtdy​−mydtdx​)=Iω、ω=(0,0,ω)

というように角運動量 L の z 成分と同じ形になっていることが分かる。そのため、最終的に回転の運動方程式は以下の通りになる。

mdt2d2r​=Fr​+fr,θ​、fr,θ​=mrω2 dtdL​=N、L=m(r×v)=Iω、N=r×F

ここまでは話を簡単にするため2次元を扱ってきたが、同様に3次元においても成立するかを見てみる。まず、直交座標系 (x,y,z) と極座標 (r,θ,ϕ) に置き換えると

x=rsinθcosϕ、y=rsinθsinϕ、z=rcosθ

であるが、これも3次元回転座標系の一例となっている。実際、まず以下の図のようにz軸を中心に角度 ϕ だけ回転させると

x′=xcosϕ+ysinϕ、y′=−xsinϕ+ycosϕ、z′=z

というような変換になる。そして、y′ 軸を中心に角度 θ だけ回転させると以下のような変換になる。

z′′=z′cosθ+x′sinθ=xsinθcosϕ+ysinθsinϕ+zcosθ x′′=−z′sinθ+x′cosθ=xcosθcosϕ+ycosθsinϕ−zsinθ y′′=y′=−xsinϕ+ycosϕ

そのため、この逆変換を求めると

x=x′′cosθcosϕ−y′′sinϕ+z′′sinθcosϕ y=x′′cosθsinϕ+y′′sinϕ+z′′sinθsinϕ z=−x′′sinθ+z′′cosθ

となることから、 x′′=rθ​=0、y′′=rϕ​=0、z′′=r とすると極座標の変換になっていることが分かる。そのため、速度なども同じ形で変換されることになる。それを踏まえたうえで、まず各成分の速度を求めると

vx​=dtdx​=dtdr​sinθcosϕ+rdtdθ​cosθcosϕ−rsinθdtdϕ​sinϕ vy​=dtdy​=dtdr​sinθsinϕ+rdtdθ​cosθsinϕ+rsinθdtdϕ​cosϕ vz​=dtdz​=dtdr​cosθ−rdtdθ​sinθ

となる。これに速度の変換式

vx​=vrθ​​cosθcosϕ−vrϕ​​sinϕ+vr​sinθcosϕ vy​=vrθ​​cosθsinϕ+vrϕ​​sinϕ+vr​sinθsinϕ vz​=−vrθ​​sinθ+vr​cosθ

を代入すると、以下の関係式が得られる。

vr​=dtdr​、vrθ​​=rdtdθ​、vrϕ​​=rsinθdtdϕ​

これらの速度は、以下のように微小変化を見ることで正しいことが分かる。

そして、加速度についても

ax​=dt2d2x​​=dtd​(dtdr​sinθcosϕ+rdtdθ​cosθcosϕ−rdtdϕ​sinθsinϕ)=dt2d2r​sinθcosϕ+2dtdr​dtdθ​cosθcosϕ−2dtdr​dtdϕ​sinθsinϕ+rdt2d2θ​cosθcosϕ−r(dtdθ​)2sinθcosϕ−2rdtdθ​dtdϕ​cosθsinϕ−rdt2d2ϕ​sinθsinϕ−r(dtdϕ​)2sinθcosϕ​ ay​=dt2d2y​​=dtd​(dtdr​sinθsinϕ+rdtdθ​cosθsinϕ+rdtdϕ​sinθcosϕ)=dt2d2r​sinθsinϕ+2dtdr​dtdθ​cosθsinϕ+2dtdr​dtdϕ​sinθcosϕ+rdt2d2θ​cosθsinϕ−r(dtdθ​)2sinθsinϕ+2rdtdθ​dtdϕ​cosθcosϕ+rdt2d2ϕ​sinθcosϕ−r(dtdϕ​)2sinθsinϕ​ az​=dt2d2z​​=dtd​(dtdr​cosθ−rdtdθ​sinθ)=dt2d2r​cosθ−2dtdr​dtdθ​sinθ−rdt2d2θ​sinθ−r(dtdθ​)2cosθ​

となるが、先ほどの速度と同じように極座標において

ax​=arθ​​cosθcosϕ−arϕ​​sinϕ+ar​sinθcosϕ ay​=arθ​​cosθsinϕ+arϕ​​sinϕ+ar​sinθsinϕ az​=−arθ​​sinθ+ar​cosθ

が成り立つため、これを代入してみると

ar​=dt2d2r​−r(dtdθ​)2−r(dtdϕ​)2sin2θ arθ​​=rdt2d2θ​+2dtdr​dtdθ​−r(dtdϕ​)2sinθcosθ arϕ​​=2dtdr​dtdϕ​sinθ+2rdtdθ​dtdϕ​cosθ+rdt2d2ϕ​sinθ

となる。そのため、極座標での各成分の運動方程式は

vr​=(dtdr​)、ωθ​=(dtdθ​)、ωϕ​=(dtdϕ​)

と置くことで、以下の通りになる。

mdtdvr​​−mrωθ2​−mrωϕ2​sin2θ=Fx​sinθcosϕ+Fy​sinθsinϕ+Fz​cosθ mrdtdωθ​​+2mvr​ωθ​−mrωϕ2​sinθcosθ=Fx​cosθcosϕ+Fy​cosθsinϕ−Fz​sinθ mrdtdωϕ​​sinθ+2mrωθ​ωϕ​cosθ+2mvr​ωϕ​sinθ=−Fx​sinϕ+Fy​cosϕ

ここで、右辺に関しては回転座標の変換（逆変換ではない）を力に置き換えた式

Fr​=Fx​sinθcosϕ+Fy​sinθsinϕ+Fz​cosθ Frθ​​=Fx​cosθcosϕ+Fy​cosθsinϕ−Fz​sinθ Frϕ​​=−Fx​sinϕ+Fy​cosϕ

と同じであることから、最終的に以下の形になる。

mdtdvr​​−mrωθ2​−mrωϕ2​sin2θ=Fr​ mrdtdωθ​​+2mvr​ωθ​−mrωϕ2​sinθcosθ=Frθ​​ mrdtdωϕ​​sinθ+2mrωθ​ωϕ​cosθ+2mvr​ωϕ​sinθ=Frϕ​​

ここで、動径方向の運動方程式を整理すると

mdtdvr​​−(mrωθ2​+mrωϕ2​sin2θ)=Fr​

となるが、括弧内の部分は遠心力となっている。実際、第１項目については、位置 r において角度 θ で回転したことによる遠心力であり、

fr,θ​=mrωθ2​

となる（動径方向にしかかからない）。第2項目については、位置 rsinθ において角度 ϕ で回転した時の遠心力であり

fx,y​=m(rsinθ)ωϕ2​

となるが、以下の図のように　x,y 平面上でしか働かないため、

fx​=fx,y​cosϕ、fy​=fx,y​sinϕ、fz​=0

というような成分を持つ力となっている。そのため、極座標においては先ほどの力の変換を用いると

fr​=fx​sinθcosϕ+fy​sinθsinϕ+fz​cosθ=fx,y​sinθ frθ​​=fx​cosθcosϕ+fy​cosθsinϕ−fz​sinθ=fx,y​cosθ frϕ​​=−fx​sinϕ+fy​cosϕ=0

というように動径方向に対して fr​ の分だけ働いていることになる。そのため、動径方向に対する運動方程式は以下の通りになる。

mdtdvr​​=Fr​+fr,θ​+fr,ϕ​、fr,θ​=mrωθ2​、fr,ϕ​=[m(rsinθ)ωϕ2​]sinθ

一方で、角度 θ に関する運動方程式は先ほどの遠心力で frθ​​ が存在することから

mrdtdωθ​​+2mvr​ωθ​=Frθ​​+fθ,ϕ​、fθ,ϕ​=m(rsinθ)ωϕ2​cosθ

として、両辺に r をかけて整理することで以下のようになる。この式は角度 θ に関する回転の運動方程式となっている。

dtdLθ​​=r(Frθ​​+fθ,ϕ​)、Lθ​=mr2ωθ​、fθ,ϕ​=[m(rsinθ)ωϕ2​]cosθ

同じ要領で角度 ϕ に関して位置 rsinθ で回転していることを踏まえて両辺に rsinθ をかけて整理してみると

dtdLϕ​​=rsinθFrϕ​​、Lϕ​=m(rsinθ)2ωϕ​

というように角度 ϕ に対する回転の運動方程式になる。ここで Lθ​ と Lϕ​ の関係式が

dtdL​=N、L=r×p、N=r×F

に対応しているか見てみよう。まず、角運動量が直交座標系において

Lx​=m(yvz​−zvy​)、Ly​=m(zvx​−xvz​)、Lz​=m(xvy​−yvx​)

となることから、極座標では以下のようになる。

Lx​=−mr2(ωθ​sinϕ+ωϕ​sinθcosθcosϕ) Ly​=mr2(ωθ​cosϕ−ωϕ​sinθcosθsinϕ) Lz​=mr2ωϕ​sin2θ

ここで、Lz​ をよく見ると Lϕ​ と同じになっていることが分かる。また、Lx​,Ly​ の両辺に sinθ をかけても Lϕ​ が出てくることから、試しに

Lx​sinθ=−Lθ​sinθsinϕ−Lϕ​cosθcosϕ Ly​sinθ=Lθ​sinθcosϕ−Lϕ​cosθsinϕ Lz​=Lϕ​

とすると、これらを整理することで

Lθ​=−Lx​sinϕ+Ly​cosϕ、Lϕ​=Lz​

という関係式が成り立つことが分かる。そこで、これを時間微分して先ほどの式が得られるかを確認してみる。まず、時間微分すると

dtdLθ​​=−dtdLx​​sinϕ−Lx​ωϕ​cosϕ+dtdLy​​cosϕ−Ly​ωϕ​sinϕ、dtdLϕ​​=dtdLz​​

となるが、Lx​,Ly​,Lz​ の時間微分

dtdLx​​=yFz​−zFy​、dtdLy​​=zFx​−xFz​、dtdLz​​=xFy​−yFx​

を代入すると以下の通りになる。

dtdLθ​​=z(Fx​cosϕ+Fy​sinϕ)−(ysinϕ+xcosϕ)Fz​−(Lx​cosϕ+Ly​sinϕ)ωϕ​ dtdLϕ​​=dtdLz​​=xFy​−yFx​

そして x,y,z を極座標の形になおすと

dtdLθ​​=r(Fx​cosθcosϕ+Fy​cosθsinϕ−Fz​sinθ)−(Lx​cosϕ+Ly​sinϕ)ωϕ​ dtdLϕ​​=dtdLz​​=rsinθ(−Fx​sinϕ+Fy​cosϕ)

であるため、以前求めた変換式から

Fx​cosθcosϕ+Fy​cosθsinϕ−Fz​sinθ=Frθ​​ −Fx​sinϕ+Fy​cosϕ=Frϕ​​ Lx​cosϕ+Ly​sinϕ=−mr2ωϕ​sinθcosθ

となる。そのため、3次元においても回転の運動方程式

dtdL​=N、L=(r×v)、N=r×F

が成り立つことが分かる。あるいは角速度 ω に関して角運動量と同じように

ωx,y​=ωx​cosϕ+ωy​sinϕ、ωθ​=ωy​cosϕ−ωx​sinϕ ωr​=ωz​cosθ+ωx,y​sinθ、ω⊥​=ωx,y​cosθ−ωz​sinθ

とすると、ωθ​ に関して Lθ​ が

Lθ​=mr2ωθ​=Ly​cosϕ−Lx​sinϕ

であることから、

ωx​=mr2Lx​​=−(ωθ​sinϕ+ωϕ​sinθcosθcosϕ) ωy​=mr2Ly​​=ωθ​cosϕ−ωϕ​sinθcosθsinϕ

であり、これを ωx,y​ の式に代入すると以下の通りになる。

ωx,y​=−ωϕ​sinθcosθ

さらに、動径方向の回転成分がないことから ωr​=0 であるため、ωr​ の式から

ωz​=ωϕ​sin2θ

であり、これを最後の式に代入すると

ω⊥​=−ωϕ​sinθ

であることになる。このことから、I=mr2 とすることで L=Iω と表されることが分かる。

　以上のように、回転運動を取り扱うには通常の運動方程式とは異なる方程式を利用する必要があることが分かる。そのため、次項で改めて物体自体の回転運動を取り扱うことにする。
